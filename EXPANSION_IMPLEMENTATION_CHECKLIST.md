### IDEA 2.2: Multi-Gen Housing + ADU Program
**Status:** Planning Phase  
**Owner:** Planning Department + Housing Consultant  
**Timeline:** 2 weeks (Zoning Audit: 1 week, Design Guide: 1 week)

**Zoning Audit:**
- [ ] Pull current El Dorado zoning code (relevant chapters: residential, accessory structures, lot subdivision, parking requirements)
- [ ] Identify barriers to ADUs (restrictions on size, footprint, setbacks, parking, owner-occupancy requirements)
- [ ] Compare to peer city ADU-friendly code (e.g., Topeka, Lawrence, Kansas City)
- [ ] Draft proposed zoning language to allow ADUs (typical: 800 sq ft max, one ADU per lot, owner must occupy one unit)

**Design Guide:**
- [ ] Collect 5–8 good ADU examples (Pinterest, case studies, real projects)
- [ ] Create visual design guide showing: duplex options, mother-in-law suite, detached ADU, garage conversion
- [ ] Include floor plans, site layouts, cost estimates (if available)
- [ ] Show financing options (ADU loans, home equity, construction financing)

**Financing Exploration:**
- [ ] Research 2–3 ADU-specific financing programs (state, federal, nonprofit)
- [ ] Interview 1–2 mortgage lenders about ADU loan terms
- [ ] Calculate ROI: typical ADU cost + rental income offset

**Community Outreach:**
- [ ] Identify 2–3 pilot neighborhoods where ADU are most feasible
- [ ] Plan community presentation on benefits (affordability, intergenerational living, rental income)

**Web Integration:**
- [ ] Add "Multi-Gen Housing" section to housing.html
- [ ] Visual design guide (infographics, floor plans)
- [ ] Zoning/permitting guide (plain language)
- [ ] Financing resource links
- [ ] "Why ADU matter" narrative

### IDEA 4.5: Remote Worker Welcome Program
**Status:** Planning Phase  
**Owner:** Economic Development + Marketing  
**Timeline:** 2 weeks (Program Design: 1 week, Execution: 1 week)

**Research Tasks:**
- [ ] Broadband coverage: Get current Velocity BEAD deployment map (addresses covered, timeline to full deployment)
- [ ] Co-working assessment: Survey existing co-working spaces in El Dorado (capacity, pricing, amenities)
- [ ] Remote work trends: Research "best small towns for remote workers" lists, demographics
- [ ] Comparable programs: Audit 3–5 small towns with remote worker programs (subsidized co-working, tax breaks, etc.)
- [ ] Remote worker survey: Post on digital nomad platforms (RemoteOK, We Work Remotely) — "Why would you move to El Dorado?" + collect responses

**Program Design:**
- [ ] Subsidized co-working: First 3 months free for relocating remote workers (apply online, simple qualification)
- [ ] Housing incentive: Partner with real estate agents on "remote worker welcome" discount/incentive (first month waived, etc.)
- [ ] Broadband guarantee: Publish coverage map + speed guarantee (gigabit by Q2 2027 via BEAD)
- [ ] Community building: Monthly remote worker meetups (networking, skill-shares, socializing)
- [ ] Slack community: Create remote-worker Slack for El Dorado (Q&A, job postings, local news)

**Marketing/Outreach:**
- [ ] Create landing page: "Remote workers wanted in El Dorado" (sell: cheap housing, fast internet, outdoor recreation, community, low cost of living)
- [ ] Outreach channels:
  - [ ] Remote work job boards (RemoteOK, We Work Remotely, FlexJobs)
  - [ ] Digital nomad communities (r/remotework, Nomadic List, Remote Workers Slack)
  - [ ] Co-working networks (WeWork pass holders, co-working communities)
  - [ ] LinkedIn targeting remote workers in large metros within 500 miles

**Cost-Benefit:**
- [ ] First-year: 20–30 remote workers relocate = $1.2M–1.8M household income (tax base: $25K–35K per person)
- [ ] Program cost: 3-month co-working subsidy (~$2K/person × 25 people = $50K/year)
- [ ] ROI: $25K+ tax base per person vs. $2K cost = 12:1 first-year ROI

**Web Integration:**
- [ ] Create dedicated "Remote Workers" landing page
- [ ] Broadband coverage map (Leaflet)
- [ ] Co-working space listings + amenities
- [ ] Community testimonials (existing remote workers in El Dorado)
- [ ] "How to move to El Dorado" guide (housing, permits, schools, recreation)
- [ ] Links to Slack, monthly meetups, application form

**Next Steps After Phase 1:**
- [ ] Recruit first cohort of remote workers
- [ ] Host first community meetup
- [ ] Track retention/satisfaction
- [ ] Measure tax base growth
