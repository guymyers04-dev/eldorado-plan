│  ├─ Yes → FDI incentives + STAR Bond + EDA

│  └─ CDFI + Bank + EDA + Property tax abatement

   └─ STAR Bond + EDA + Bank + Developer equity

## QUICK REFERENCE: "GO/NO-GO" QUESTIONS FOR EACH MECHANISM

### Before pursuing a funding source, ask:

**1. Is the timeline acceptable?**
- If need capital in <6 months: Special districts, direct donations, bank loans
- If 6–12 months acceptable: Bonds, CDFI loans, smaller grants
- If 12+ months acceptable: EB-5, FDI, LIHTC, Historic Tax Credits

**2. Are there regulatory/compliance barriers?**
- EB-5: Requires immigration lawyer, regional center, USCIS approval (complex)
- HTC: Requires NRHP listing, SHPO approval (6–9 months)
- LIHTC: Requires affordability covenants for 20+ years (permanent restriction)
- PPP: Requires competitive RFP, complex contracting

**3. Is the project "fundable" by this source?**
- Grants: Project must align with program mission (federal, state, foundation)
- Private debt: Project must have strong cash flow/ROI (50%+ of capital-intensive projects fail this test)
- Equity/Impact capital: Project must have social/environmental impact

**4. What's the cost (fees, dilution)?**
- Grants: $0 cost (non-dilutive)
- Bank loans: 5–7% interest rate
- CDFI: 4–6% + 1–2% fees
- EB-5: 1.5–2.5% placement fees + regional center operator fees
- Private equity: 25–40% dilution of future returns (high cost)

**5. Does the project have sufficient equity/skin-in-game?**
- Most lenders require 15–30% developer equity (not borrowed)
- EB-5 projects more flexible (investors tolerate 100% capital structure)
- Grants typically require 10–20% local match
