Vision 2040 Growth Catalysts

Key Drivers of Tax Base Expansion

Vision 2040 accelerates tax base growth through strategic initiatives across six core areas. Each driver adds commercial/industrial property value, sales tax revenue, and employment tax contributions.

1. Waterfront Development

Marina complex, waterfront dining, boutique resort, retail. Commercial value concentration on highest-use lakefront land.

Est. Value Add: $45-55M commercial property

2. North Gateway District

Mixed-use neighborhood with 1,000+ residential units + 200K sf retail/office. Highest density = highest tax value per acre.

Est. Value Add: $120-150M (residential+commercial)

3. Industrial Mega-Site

500+ acres + infrastructure attracts manufacturing, distribution, advanced technology. Generates sales tax + property tax.

Est. Value Add: $85-110M industrial property

4. Tourism & Hospitality

Restaurants, lodging, attractions. Drives sales tax revenue directly (10% of visitor spending) + property value.

Est. Value Add: $35-45M commercial property

5. Employment Growth

Higher-wage jobs attract workers + families. Supports 200-300 new households + supporting commercial activity.

Est. Value Add: $60-80M residential property

6. Regional Integration

Position El Dorado as regional hub for Wichita MSA. Spillover commercial development + regional event hosting.

Est. Value Add: $25-35M commercial property
