Integration Instructions

Option 1: Google Forms (Easiest)

1. Create a Google Form: https://forms.google.com
2. Add fields:
   - Company Name (Text)
   - Your Name (Text)
   - Email Address (Email)
   - Phone (Text)
   - Interest Level / Role (Multiple Choice)
   - Message (Paragraph)
3. Get embed code: Click Share → </> (Embed)
4. Paste into each industry page HTML
5. Response sheet auto-populates: Google Sheets integration

Option 2: Formspree (Free Email Submissions)

1. Go to: https://formspree.io
2. Create project
3. Add email: yourname@eldorado.gov
4. Get form action URL: https://formspree.io/f/[ID]
5. Replace form action in HTML below with your URL
6. Submissions go to your email

Option 3: Custom Backend (Most Control)

1. Set up form handler (Node.js, Python Flask, PHP, etc.)
2. Connect to CRM or database
3. Advanced tracking + automation
4. More complex but most powerful

Quick HTML Template (Formspree)

<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
  <div class="form-group">
    <label for="company">Company Name *</label>
    <input type="text" id="company" name="company" required>
  </div>

  <div class="form-group">
    <label for="email">Email Address *</label>
    <input type="email" id="email" name="email" required>
  </div>

  <button type="submit">Send Inquiry →</button>
</form>
