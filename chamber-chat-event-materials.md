## CRISIS MANAGEMENT

### If Projector Fails
→ Switch to printed slides + verbal presentation  
→ Stay composed, make eye contact, keep talking

### If You Lose Your Place in Notes
→ Pause, smile, look at slide ("Great question embedded in the slide here...")  
→ Take a sip of water, refocus

### If Someone Asks a Hard Question
→ Answer honestly: "That's a great question. Let me follow up with you after..."  
→ Write it down, add to follow-up list

### If You Run Out of Time
→ Speed through last 1–2 slides, hit the ask  
→ Save detailed Q&A for follow-up calls

### If Too Few / Too Many People Show Up
→ Same presentation, adjust tone (more intimate if small; more energetic if large)  
→ Still accomplish the ask (sign-ups)
