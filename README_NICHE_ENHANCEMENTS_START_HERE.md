## WHAT IS THIS?

This project adds 22 interactive tools and features to the El Dorado Vision 2040 website across 5 content categories:
- **Economy** (3 ideas): Milestone timeline, success stories, job calculator
- **Housing** (3 ideas): Affordability tracker, commute finder, resident personas
- **Place** (4 ideas): Activity calendar, before/after comparisons, walking tour, artist board
- **People** (2 ideas): Education pathways, public safety map
- **Build** (2 ideas): Infrastructure tracker, trail map
- **Finance** (2 ideas): Tax calculator, funding tracker
- **Community** (3 ideas): Community champions, feedback loop, event calendar
- **Region** (2 ideas): MSA dashboard, commute/migration map
- **Research** (2 ideas): Sentiment survey, data dashboard

**Each tool addresses a core resident concern and makes abstract Vision 2040 concepts concrete.**

## FREQUENTLY ASKED QUESTIONS

**Q: Can we start before Phase 1 is done?**  
A: Not recommended. Phase 1 establishes design system, processes, team rhythm. Rushing skips these foundations and creates debt later.

**Q: Can we do all 22 at once?**  
A: No. Team would be scattered, quality would suffer, city coordination would break down. The phased approach forces priorities + quality.

**Q: What if we run out of time?**  
A: The roadmap has 20 weeks = 4 months. If you have 6+ months, you're ahead. Build more. If you have 2 months, drop Phases 5-6 (regional + measurement) and launch Phase 4 as MVP. Core tools (Timeline, Before/After, Champions, Tax Calc, Affordability, Commute, Pathways, Infrastructure, Trails) hit 80% of impact.

**Q: Who owns this after launch?**  
A: City manager should assign ongoing owner (1 FTE or 0.5 FTE contractor) for:
- Monthly data updates (funding tracker, infrastructure status)
- Quarterly feedback reports (C2)
- Annual survey (Res1)
- Quarterly data dashboard updates (Res2)

**Q: Do we need all 22?**  
A: No. Minimum viable set for launch (MVP):
- Phase 1: Timeline, Before/After, Tax Calc (3 tools)
- Phase 2: Affordability, Commute, Pathways (3 tools)
- Phase 4: Infrastructure, Trails (2 tools)
- **MVP = 8 tools addressing core resident concerns** (180-240 hours)
- Phases 2-3 optional: success stories, job calc, personas, artist board
- Phase 5-6: Nice to have (regional context, measurement infrastructure)
