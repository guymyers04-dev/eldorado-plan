## 🔍 COMMON ISSUES TO WATCH FOR

### Spelling & Grammar
- [ ] "El Dorado" (not "El dorado" or "el Dorado")
- [ ] "Vision 2040" (capitalize both)
- [ ] "Kansas" (always capitalized)
- [ ] Consistent hyphenation (e.g., "year-over-year")
- [ ] Number consistency (spell out 1-9, use numerals 10+)
- [ ] Tense consistency (don't mix past/present)

### Tone & Clarity
- [ ] "Leverage" → Use concrete verb
- [ ] "Utilize" → Use "use"
- [ ] Passive voice → Convert to active where possible
- [ ] Jargon → Define on first mention or replace with plain language
- [ ] Wall of text → Break into shorter paragraphs
- [ ] Vague language → Be specific with numbers/dates

### CTAs (Critical for Conversion)
❌ Weak:
- "Click here"
- "Learn more"
- "Read on"
- "Get started"

✅ Strong:
- "Explore 16 Development Properties"
- "View Property Details & ROI Projections"
- "Download Investor Brief"
- "Schedule Property Tour"
- "Calculate Your Potential Return"

### Terminology Consistency

**Use consistently throughout:**
- "El Dorado" (not "the city" or "our community" repeatedly)
- "Vision 2040" (not "the vision" or "2040 plan")
- "Investors" (not "you" or "potential investors" - pick one)
- "Residents" (not "community members" - be consistent)
- "Property" / "development site" (don't mix)
- "Workforce development" (not "jobs training" or "education")

## 🛠️ TOOLS

### For Copy Editing:
1. **Grammarly** (browser extension) — Real-time grammar check
2. **Hemingway Editor** (hemingwayapp.com) — Readability check
3. **VSCode Find/Replace** — Bulk fix common issues
4. **Read-aloud** (VSCode extension) — Catch awkward phrasing

### Quick Fixes with Find/Replace:
```
Find: "  " (double space) → Replace: " " (single)
Find: " ," → Replace: ","
Find: " ." → Replace: "."
Find: "click here" → Replace: [specific CTA]
Find: "learn more" → Replace: [specific CTA]
Find: "el dorado" → Replace: "El Dorado"
```
