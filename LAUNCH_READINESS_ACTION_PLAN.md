## 🛠️ COMMON ISSUES & QUICK FIXES

### If Analytics Aren't Tracking
1. Check GA4 property ID is correctly placed in `<head>` of all pages
2. Verify GA4 account has correct IP address in allowlist
3. Wait 24 hours for data to begin appearing
4. Check GA4 Real-Time view immediately after visiting site

### If Page Loads Slowly
1. Enable gzip compression on web server
2. Verify CSS/JS files are minified
3. Check image file sizes (<200KB each)
4. Enable browser caching headers

### If Mobile Layout Looks Wrong
1. Clear browser cache (Ctrl+Shift+Del)
2. Verify viewport meta tag is in `<head>`
3. Check CSS media queries are loading correctly
4. Test in Chrome DevTools responsive mode

### If Navigation Isn't Working
1. Check `js/nav.js` is properly linked
2. Verify dropdown menus have proper ARIA attributes
3. Check CSS z-index values for dropdowns
4. Test keyboard navigation (Tab key)
