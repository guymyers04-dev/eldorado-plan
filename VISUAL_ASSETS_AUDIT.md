## Photo Specifications Guide

### Standard Dimensions

| Use Case | Dimensions | Aspect Ratio | Format |
|----------|-----------|--------------|--------|
| Hero slides (homepage) | 800×600px | 4:3 | JPG/WebP |
| Gallery cards (place pages) | 600×340px | 16:9 | JPG/WebP |
| Construction gallery | 500×375px | 4:3 | JPG/WebP |
| Investor pitch deck | 1200×800px | 3:2 | JPG |
| Mobile optimized | 400×300px | 4:3 | JPG/WebP |

### Technical Standards

- **Format:** JPG (primary), WebP (for modern browsers)
- **File Size:** <150KB per image (gzip)
- **Color Profile:** sRGB
- **Optimization:** TinyPNG or ImageOptim
- **Accessibility:** Descriptive alt text (80 characters max)

## Implementation Workflow

### When Photos Arrive:

1. **Organize by Category** (Excel/Airtable)
   - Downtown (3 photos)
   - Lake (3 photos)
   - Residential (2 photos)
   - Industrial (2 photos)
   - etc.

2. **Optimize Images**
   ```bash
   # Resize to spec dimensions
   convert photo.jpg -resize 800x600 -quality 85 hero-1.jpg
   
   # Generate WebP format
   cwebp photo.jpg -o photo.webp -q 85
   ```

3. **Replace Unsplash URLs** (sed bulk replace)
   ```bash
   # Replace in development-finder.html
   sed -i 's|https://images.unsplash.com/[^"]*|/images/local/|g' economy.html
   ```

4. **Upload to CDN** (Cloudinary or similar)
   - Store organized folders: `/images/downtown/`, `/images/lake/`, etc.
   - Enable automatic responsive image resizing

5. **Update HTML**
   ```html
   <!-- Old -->
   <img src="https://images.unsplash.com/photo-xxx?w=800" alt="Downtown">
   
   <!-- New -->
   <img src="/images/downtown/main-street-day.jpg" alt="El Dorado Downtown Main Street">
   ```

6. **Test & Deploy**
   - Visual regression testing (Percy.io or similar)
   - Mobile responsiveness check
   - Lighthouse performance validation
   - Deploy to production
