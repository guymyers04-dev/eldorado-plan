## TROUBLESHOOTING GUIDE

### "Email bounced — address not found"
```
☐ Double-check email address on company website
☐ Try: info@[company].com as fallback
☐ Try: LinkedIn to find correct contact
☐ Skip and move to next company if unresolvable
```

### "UTM link is too long / looks broken in email"
```
☐ Use URL shortener: bit.ly or tinyurl.com
☐ Create short link: bit.ly/eldorado-precision-ag
☐ Use in email instead of full long link
☐ Still tracks (shortener redirects to full UTM URL)
```

### "Lead form not working"
```
☐ Check: Form embedded code is correct
☐ Check: No HTML syntax errors
☐ Test: Submit form yourself
☐ Check: Google Forms response sheet has entries
☐ Fallback: Create simple email signup instead
```

### "Analytics not showing traffic"
```
☐ Wait 24 hours (GA4 has delay)
☐ Check: GA4 code is installed (use DevTools)
☐ Check: Not in Incognito window (won't track)
☐ Fallback: Check email click-through in Gmail instead
```
