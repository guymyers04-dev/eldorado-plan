# El Dorado Vision 2040 — RAPID RESPONSE PLAYBOOKS

**Quick-reference decision trees for common risk scenarios**

## PLAYBOOK 1: DATA CENTER PROSPECT LOST

**Trigger:** Prospect declines or chooses competitor

**IMMEDIATE (24 hours)**
```
EDA CEO call to prospect:
├─ Thank them for consideration
├─ Ask: "What was the deciding factor?"
├─ Ask: "Would circumstances change if X improved?"
└─ Document response
    
Communications Manager:
├─ Prepare competitor analysis (who won? what did they offer?)
└─ Start prospect #2–5 outreach acceleration
```

**SHORT-TERM (1 week)**
```
Data Center Director decision tree:

Is this the 1st prospect lost?
├─ YES → Continue recruitment (normal operation)
│   └─ Accelerate prospects #2–5 outreach
│
└─ NO (2+ prospects lost to same competitor)
    ├─ Was incentive offer decisive?
    │  ├─ YES → Escalate to EDA Board (increase incentives?)
    │  └─ NO → Evaluate DC Director performance
    │
    └─ Was competitor's water/infrastructure advantage decisive?
       ├─ YES → Revisit water rights strategy
       └─ NO → Evaluate DC Director fit; consider replacement
```

**MEDIUM-TERM (1 month)**
```
Post-mortem meeting:
├─ Consultant review (is recruitment strategy sound?)
├─ Competitor analysis (what's their real advantage?)
├─ CBRE/JLL performance review (are they delivering right prospects?)
├─ DC Director performance assessment
│  ├─ Has strong relationships? (Stay)
│  └─ Lacks relationships? (Consider replacement)
│
└─ Action items:
    ├─ Adjust pitch (address competitor advantage)
    ├─ Evaluate incentive changes
    └─ Accelerate prospects #2–5
```

**CONTINGENCY TRIGGERS**
```
If 2+ prospects lost to same competitor:
├─ Competitor analysis: What's their winning advantage?
├─ Decision: Match competitor or pivot sector?
│  ├─ Match → Increase incentives or improve facilities
│  └─ Pivot → Shift to FDI or alternative industrial user
│
└─ If DC Director underperforming:
    ├─ Can director improve? (coaching, resources)
    └─ Otherwise → Replace by Month 3–4

If no qualified prospects by Month 4:
└─ Shift strategy entirely:
    ├─ FDI acceleration (target 3–4 instead of 2)
    ├─ Alternative industrial user (beverage, chemicals)
    └─ OR accept data center delayed to 2028–2029
```

## PLAYBOOK 2: RECESSION APPEARS IMMINENT

**Trigger:** Yield curve inverts; Fed signals pause; unemployment rising; media recession concerns

**IMMEDIATE (Week 1)**
```
EDA CEO actions:
├─ Brief City Commission (prepare for revenue risk)
├─ Assess municipal cash reserves (do we have 6-month buffer?)
├─ Shift messaging: "Countercyclical advantage" (cloud spending, housing shortage)
│
Data Center Director:
├─ Shift from "relationship building" to "expedited closes"
├─ All prospect calls: "Timeline accelerated to Q4 2026" (close before market freezes)
│
Housing Consultant:
├─ Prioritize LIHTC/USDA close (lock in funding before credit market tightens)
└─ Accelerate site acquisition (land cheaper in recession; buy before prices drop)
```

**SHORT-TERM (1 month)**
```
Scenario planning:
├─ "Recession playbook": What gets cut?
│  ├─ Marketing/video production: Defer to Q2 2027
│  ├─ Travel: Reduce 50% (focus on highest-priority prospects)
│  └─ Staffing: No new hires; maintain current 4-person team
│
├─ Financial planning:
│  ├─ Reduce Year 2 budget from $510K to $350K
│  └─ Defer non-critical initiatives
│
└─ Messaging:
    ├─ Communicate to prospects: "Data center recruitment accelerating; capitalize on lower costs"
    └─ Communicate to residents: "Steady growth strategy; new facilities create recession-proof jobs"
```

**MEDIUM-TERM (Ongoing through recession)**
```
Execution phase:

Maintain team integrity:
├─ NO LAYOFFS (signal stability to employees & community)
├─ Maintain salaries (don't cut compensation)
└─ Acknowledge stress (wellness programs, flexible scheduling)

Shift to "relationship maintenance":
├─ Existing prospects: Weekly check-ins (keep deal momentum)
├─ Existing partners: Regular communication (maintain relationships)
└─ Reduce new prospect acquisition (too much noise)

Prepare for recovery:
├─ Pre-position projects (permits, designs ready to execute)
├─ Maintain broker relationships (CBRE/JLL relationships live)
└─ Have "surge plan" ready (when recession ends, accelerate hiring/spending)

Quarterly reassessment:
└─ Is recession deepening or moderating?
    ├─ Deepening → Further cost cuts; extend timelines
    └─ Moderating → Begin recovery positioning
```

**SUCCESS CRITERIA (Recession Playbook)**
```
Phase 1 Success Redefined for Recession:
├─ Staffing: Team intact; morale stable
├─ Prospects: 1+ LOI signed (even if delayed execution)
├─ Housing: Financing secured (groundbreaking delayed to 2028 OK)
├─ Regional: Community confidence maintained
└─ Financial: Phase 2 budget secured for 2028 recovery
```

## PLAYBOOK 3: HOUSING FINANCING DELAYED/REJECTED

**Trigger:** LIHTC syndicator indicates delay; USDA RD tightens underwriting; application rejected

**IMMEDIATE (Week 1)**
```
Housing Consultant actions:

Contact primary funders:
├─ LIHTC syndicator → "What caused delay? What's new timeline?"
└─ USDA RD → "Are underwriting standards changing? What do you need from us?"

Identify alternates:
├─ Secondary LIHTC syndicators (identify 2–3 backup syndicators immediately)
├─ State programs: KHIF, KDHE (research eligibility)
└─ Private EB-5 investors (reach out to Kansas Regional Center)

Timeline adjustment:
└─ Communicate revised timeline to EDA CEO & team
```

**SHORT-TERM (1 month)**
```
Decision tree:

Is primary LIHTC available (just delayed)?
├─ YES → Accept delayed timeline; groundbreaking Q3/Q4 2027 vs. Q2
│
└─ NO (rejected or uncertain)
    ├─ Contact secondary syndicators (submit applications to 2–3 alternatives)
    ├─ Research state programs (KHIF equivalent funding?)
    └─ Evaluate private/EB-5 funding option
```

**REVISED PROJECT PARAMETERS**
```
If LIHTC timing uncertain:

Option A: Reduce unit count
├─ Phase 1A: 100 units (vs. 150)
├─ Phase 1B: 50 units (defer to 2028)
└─ Financing need: $6–8M (vs. $8–12M)

Option B: Alternative financing mix
├─ USDA RD: $4–6M (primary debt)
├─ State programs: $1–2M (grants)
├─ EB-5: $1–2M (equity)
├─ Local TIF: $1M
└─ Total: $8–12M (alternative to LIHTC)

Option C: Hybrid model
├─ USDA RD loan (fast-track)
├─ Reduced unit count initially
└─ Add LIHTC phase 2 when funding available
```

**CONTINGENCY THRESHOLDS**
```
Green Light (Continue as planned):
└─ LIHTC pre-approval by December 2026

Yellow Light (Modified plan):
├─ LIHTC delayed to Q1 2027
├─ Groundbreaking shifted to Q3 2027
├─ Unit count unchanged (150)
└─ Financing mix adjusted (USDA heavy)

Red Light (Major adjustment):
├─ LIHTC unavailable or rejected
├─ Activate "Option B" (alternative financing)
├─ Reduce unit count to 100
└─ Groundbreaking to Q4 2027 (still Phase 1)

Hard Stop (Defer to Phase 2):
├─ No financing available
├─ Defer housing to 2028 Q2+ (after market stabilizes)
└─ Focus Year 1 on data center/FDI/lake activation
```

## PLAYBOOK 4: CRITICAL STAFF RESIGNATION

**Trigger:** Data Center Director or FDI Officer announces departure

**IMMEDIATE (24 hours)**
```
EDA CEO:
├─ Exit interview (understand reasons; knowledge transfer)
├─ Offer consulting extension (3 months at $3K/month; knowledge handoff)
├─ Assess prospect pipeline hand-off
│
Data Center Director (if leaving):
├─ Introduction calls to top 3–5 prospects (EDA CEO + departing director; reassure continuity)
└─ Documentation: Prospect status, broker relationships, next steps
│
FDI Officer (if leaving):
├─ JETRO/GTAI/KOTRA relationship hand-off calls
└─ FDI prospect pipeline documentation

Communications Manager:
└─ Draft internal & external communications (all staff, board, partners; emphasize continuity)
```

**SHORT-TERM (Week 1)**
```
Interim coverage decision tree:

Is resignation >8 weeks before critical milestone?
├─ YES → Hire replacement immediately (normal recruitment)
│
└─ NO (critical deadline <8 weeks)
    ├─ Option A: Interim consultant ($3K/month; backfill role)
    │  └─ Hire replacement in parallel (3-month overlap)
    │
    └─ Option B: EDA CEO assumes role temporarily
       ├─ DC Director duties → EDA CEO or contract CBRE broker
       ├─ FDI Officer duties → EDA CEO or interim consultant
       └─ Max 4–6 weeks (then hire replacement)
```

**MEDIUM-TERM (1 month+)**
```
Replacement recruitment:

Expedited process (target 4–6 week hire):
├─ Executive recruiter already on retainer (reactivate)
├─ Job posting: Multi-platform (Indeed, LinkedIn, ICDC, IEDC)
├─ Interviews: Compressed timeline (2 rounds instead of 3)
├─ Onboarding: Parallel with departing staff (2-week overlap if possible)

During gap period:
├─ Maintain prospect relationships (EDA CEO + interim staff)
├─ Continue recruitment (momentum critical)
├─ Communicate stability (reassure partners & prospects)
└─ Prepare for handoff (documented prospect briefings, market intel)
```

**MORALE MANAGEMENT**
```
Remaining team:
├─ Group meeting: "Why they left is normal; we're stable"
├─ Individual check-ins: "How are you feeling? Support available"
├─ Workload assessment: "Do we need interim help for your role?"
│
Compensation review:
├─ Consider spot bonuses for remaining staff (recognize stability)
├─ Offer additional PTO (acknowledge stress)
└─ Confirm career path confidence

Quarterly pulse surveys:
└─ Post-incident check-in: Team morale, satisfaction, concerns
```

## PLAYBOOK 5: BOARD GRIDLOCK / CONFLICT

**Trigger:** EDA Board members publicly disagree on priorities; decisions delayed; staff receives conflicting direction

**IMMEDIATE (1 week)**
```
Board Chair + EDA CEO:

1. Identify source of conflict:
   ├─ Strategic disagreement (data center vs. FDI priority)?
   ├─ Risk tolerance (aggressive vs. conservative)?
   ├─ Fiscal concern (worried about budget)?
   └─ Sector interest (board member benefits more from X vs. Y)?

2. Executive session:
   ├─ All board members present
   ├─ Confidential discussion (no public record)
   └─ Goal: Identify shared ground & resolve concerns

3. Clear decision protocol:
   ├─ Board establishes decision-making process (what requires board vote vs. CEO discretion)
   ├─ Board establishes approval thresholds (unanimous vs. majority)
   └─ Board establishes communication protocol (Board Chair communicates board decision to staff)
```

**SHORT-TERM (2–4 weeks)**
```
Conflict resolution options:

Option A: Consensus building (preferred)
├─ Board Chair facilitates series of 1-on-1 conversations
├─ Identify core concerns of each board member
└─ Find solution that addresses all concerns

Option B: Compromise voting
├─ Board votes on priority ordering
├─ Minority views documented for record
├─ Commitment from all members to support voted decision

Option C: Escalate to City Commission
├─ If board remains gridlocked (can't reach consensus in 2 weeks)
├─ City Commission asked to intervene (rewrite board charter if needed)
└─ Possible board restructuring (add/remove members; change voting rules)

Option D: Clarify board role
├─ Renegotiate board charter (what is board's actual decision authority?)
├─ Establish clear boundaries (board approves strategy; CEO executes)
└─ Empower CEO to execute with quarterly board updates (not approval)
```

**COMMUNICATION TO STAFF**
```
All-hands meeting (confidential):
├─ EDA CEO: "Board has resolved differences; we have unified direction"
├─ Reiterate priorities (data center, FDI, housing, lake, website)
├─ Confirm: "I have authority to execute; I will keep you informed"
└─ Encourage: "Ask me if you receive conflicting direction; I'll clarify with board"
```

**PREVENTION (Going forward)**
```
Monthly board meetings:
├─ Structured agenda (not ad-hoc)
├─ Monthly metrics review (shared understanding of progress)
├─ Strategic briefing (market conditions, competitive landscape)
├─ Q&A: All board concerns addressed at meeting (not offline)
│
Quarterly strategy review:
├─ Deep dive on Phase 1 progress vs. plan
├─ Risk assessment update
├─ Stakeholder feedback
└─ Adjust strategy if needed (transparent decision-making)

Maintain alignment:
└─ Board satisfaction survey (quarterly)
    ├─ "Do you feel heard?"
    ├─ "Are you confident in strategy?"
    └─ "Do you trust leadership?"
```

## PLAYBOOK 6: FEDERAL BUDGET CUTS / GRANT DEFUNDING

**Trigger:** Congress cuts EDA/HUD/USDA programs; BEAD funding at risk; grant sources reduce

**IMMEDIATE (1 week)**
```
Identify impact:
├─ Which programs affect El Dorado? (BEAD, EDA, USDA RD, HUD)
├─ What's the funding amount at risk? ($5M? $20M? $100M+?)
└─ What's the timeline? (When is funding cut effective?)

Mitigation assessment:
├─ Can we close existing applications before cuts take effect?
├─ Are there grandfather clauses (projects in process exempt)?
└─ Can we shift to alternative federal programs?
```

**SHORT-TERM (2–4 weeks)**
```
Diversification planning:

Immediate actions:
├─ Housing: Accelerate LIHTC/USDA close (lock in before cuts)
├─ Broadband: Track BEAD deployment status (if funded, accelerate rollout)
└─ Water: Research alternative federal sources (USACE, Corps of Engineers)

Alternative funding sources:
├─ State programs: KHIF (housing), KDHE, Kansas Commerce grants
├─ Municipal bonds: Assess city bonding capacity (infrastructure financing)
├─ Private investment: EB-5 visas, private equity, venture capital
├─ Public-private partnerships: Infrastructure privatization models

Advocacy:
└─ Kansas delegation contact:
    ├─ Communicate El Dorado priorities
    ├─ Request support for EDA/HUD/USDA programs
    └─ Seek state-level support if federal cuts occur
```

**FINANCIAL IMPACT ASSESSMENT**
```
Scenario: 20% federal funding reduction

Housing:
├─ LIHTC: Assume 10–15% reduction in equity available
├─ USDA RD: Assume tighter underwriting (higher equity requirement)
├─ Result: Reduce Phase 1 to 100 units vs. 150
│
Broadband:
├─ BEAD: Assume partial delay (deployment 6–12 months slower)
├─ Result: Still live by 2027; modest impact on data center recruitment
│
Water infrastructure:
├─ USACE: Assume 25% reduction
├─ Local share increases $3M–$5M
│
Alternative: Pursue state programs + municipal bonding to offset

Overall: Achievable but requires contingency funding & timeline extensions
```

**CONTINGENCY TRIGGERS**
```
Green Light (>75% federal funding secure):
└─ Continue Phase 1 as planned

Yellow Light (50–75% federal funding):
├─ Activate alternative sources (state, bonds, private)
├─ Reduce Phase 1 scope (100 units vs. 150)
└─ Extend timelines (groundbreaking Q4 2027 vs. Q2)

Red Light (<50% federal funding):
├─ Major Phase 1 revision:
├─ Housing: Defer to Phase 2 (2028+)
├─ Infrastructure: Pursue municipal bonding only
├─ Broadband: Accept delayed BEAD deployment
│
└─ Focus Year 1 on data center/FDI (less dependent on federal funding)

Hard Stop (>80% federal funding lost):
└─ Pause Phase 1; wait for budget environment to stabilize (2028+)
```
